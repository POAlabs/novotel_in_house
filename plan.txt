To help you transition this "HUB OS" concept from the current React prototype into Flutter, you'll need to replicate the specific architectural logic and design system we've built.

Here is a detailed explanation of the implementation details for your AI editor:

### 1. Typography & Visual Identity

* **Font Integration:** The entire app uses the **Sora** font from Google Fonts. It is a geometric sans-serif that requires specific weighting (Bold and ExtraBold) and slightly increased letter-spacing to achieve the modern, tech-forward look.
* **Color Palette:** The UI is built on a clean White and Slate-50 background. The "Diagnostic" states are binary: **Emerald Green** (representing 100% operational/safe) and **Bright Red** (representing a breach or active issue). There are no "warning" yellows; everything is either stable or critical.
* **Corner Radii:** Use a "Super-ellipse" aesthetic. Containers, cards, and buttons use high border-radius values (ranging from 24.0 to 48.0 logical pixels) to create a premium, hardware-like feel.

### 2. Navigation Architecture

* **Left-Hand Navigation Rail:** Instead of a bottom navigation bar, the app uses a slim, fixed sidebar. It contains the primary view toggles (Home and Sections) and a settings gear at the bottom.
* **The Floor Switcher:** This is a key featureâ€”a vertical scrolling list of square floor indicators within the sidebar. These indicators must change color based on the floor's status: Green for safe floors, Red for floors with active issues. When a floor is selected, it becomes solid colored with a shadow; when unselected, it shows as a light tinted border.

### 3. Dashboard Logic (Central Command)

* **Integrity Metric:** This is a large, circular progress indicator. The "Health" percentage is calculated dynamically: start at 100% and subtract a fixed percentage (e.g., 5%) for every active issue. If the health is above 80%, the ring is green; if below, the ring and the "Stable/Critical" badge turn red.
* **Metric Cards:** Use large, bold numbers with tiny, all-caps, wide-spaced labels underneath. Each metric (Urgent, Pending, Uptime) has its own tinted icon background.
* **Live Diagnostics Feed:** A list of recent issues that filters based on state. Each card shows the issue description, department, floor number, and a priority indicator.

### 4. The "Sections" View (Interactive Floor Map)

* **Adaptive Grid:** This view switches based on the floor's contents. For floors 1-9, it displays a responsive grid of "Room Cards" (e.g., Room 1101, 1102). For specialized floors (Basements, Ground, 10/11), it displays larger "Area Cards" (e.g., Kitchen, Ballroom).
* **Room State Logic:** Each card is a diagnostic tool. If a room has an issue tied to its ID in the database, the card turns red and displays an alert icon. If no issue exists, it remains emerald green with a subtle checkmark.

### 5. Interaction & Motion

* **Micro-interactions:** Buttons should have a slight "active" scale-down effect (97% scale) when tapped to feel responsive.
* **Entrance Animations:** When switching between floors or tabs, the new content should perform a subtle "Slide and Fade" animation from the bottom.
* **Diagnostic Pulse:** Any element in a "Red/Critical" state should have a slow pulse animation to draw the administrator's attention.

### 6. Data Modeling

* **Issues Object:** Needs to track ID, Floor, Area (Room or Section), Description, Status (Ongoing/Completed), Priority (Urgent/High/Medium/Low), Department, and a Timestamp.
* **Floor Map:** A list defining each floor's ID and its specific sub-areas (since floors G, B1-B3, 10, and 11 have custom names instead of room numbers).

### 7. Global Header

* **Contextual Title:** The header text should update dynamically to show "Central Command" on the home tab, or "Floor [ID]" when browsing specific sections.
* **Identity Block:** A small profile section on the right showing the user's name and role in all-caps tracking.






import React, { useState, useMemo } from 'react';
import { 
  Building2, 
  AlertTriangle, 
  CheckCircle2, 
  Plus, 
  Settings, 
  User, 
  ChevronRight, 
  MapPin,
  Home,
  LogOut,
  Activity,
  Layers,
  Zap,
  ShieldAlert,
  Cpu
} from 'lucide-react';

const App = () => {
  const [selectedFloor, setSelectedFloor] = useState('11');
  const [isReporting, setIsReporting] = useState(false);
  const [activeTab, setActiveTab] = useState('master_logs'); 
  
  const user = {
    name: "Alex Sterling",
    role: "Administrator"
  };

  const floors = [
    { id: '11', name: '11th Floor', areas: ['TnT', 'Kitchen'] },
    { id: '10', name: '10th Floor', areas: ['Kitchen', 'Executive Lounge', 'Pool Bar', 'Swimming Pool', 'Guest Rooms'] },
    { id: '9', name: '9th Floor', areas: [] },
    { id: '8', name: '8th Floor', areas: [] },
    { id: '7', name: '7th Floor', areas: [] },
    { id: '6', name: '6th Floor', areas: [] },
    { id: '5', name: '5th Floor', areas: [] },
    { id: '4', name: '4th Floor', areas: [] },
    { id: '3', name: '3rd Floor', areas: [] },
    { id: '2', name: '2nd Floor', areas: [] },
    { id: '1', name: '1st Floor', areas: [] },
    { id: 'G', name: 'Ground Floor', areas: ["Gemma's", 'Main Kitchen', 'Social Hub', 'Front Office', 'Simba Ballroom'] },
    { id: 'B1', name: 'Basement 1', areas: ['Back Office', 'Finance', 'Staff Cafeteria', 'Parking'] },
    { id: 'B2', name: 'Basement 2', areas: ['Parking', 'Bakery', 'Control Room', 'Laundry'] },
    { id: 'B3', name: 'Basement 3', areas: ['Engineering Workshop', 'Stores', 'Parking'] },
  ];

  const mockIssues = [
    { id: 1, floor: '10', area: 'Pool Bar', desc: 'Shattered glass near deck', status: 'Ongoing', priority: 'High', dept: 'Security', time: '12m ago' },
    { id: 2, floor: 'B3', area: 'Engineering', desc: 'Main HVAC Compressor Leak', status: 'Ongoing', priority: 'Urgent', dept: 'Engineering', time: '45m ago' },
    { id: 3, floor: 'G', area: 'Front Office', desc: 'Check-in Kiosk offline', status: 'Completed', priority: 'Medium', dept: 'IT', time: '2h ago' },
    { id: 4, floor: '11', area: 'Kitchen', desc: 'Freezer Temp Drop (-4C)', status: 'Ongoing', priority: 'High', dept: 'Engineering', time: '1h ago' },
    { id: 5, floor: '5', area: 'Room 512', desc: 'Electronic safe malfunction', status: 'Ongoing', priority: 'Low', dept: 'Engineering', time: '3h ago' },
  ];

  const getFloorStatus = (floorId) => {
    return mockIssues.some(issue => issue.floor === floorId && issue.status === 'Ongoing');
  };

  const currentFloorIssues = useMemo(() => {
    return mockIssues.filter(iss => iss.floor === selectedFloor && iss.status === 'Ongoing');
  }, [selectedFloor]);

  const metrics = useMemo(() => {
    const ongoing = mockIssues.filter(i => i.status === 'Ongoing');
    const urgent = ongoing.filter(i => i.priority === 'Urgent' || i.priority === 'High').length;
    const health = 100 - (ongoing.length * 5); 
    return {
      total: ongoing.length,
      urgent: urgent,
      health: Math.max(health, 0)
    };
  }, [mockIssues]);

  const floorRooms = useMemo(() => {
    const floorNum = parseInt(selectedFloor);
    if (isNaN(floorNum) || floorNum < 1 || floorNum > 9) return [];
    const rooms = [];
    for (let i = 1; i <= 40; i++) {
      const roomNum = `${floorNum}${i.toString().padStart(2, '0')}`;
      rooms.push({
        id: roomNum,
        hasIssue: mockIssues.some(iss => iss.area === `Room ${roomNum}` && iss.status === 'Ongoing')
      });
    }
    return rooms;
  }, [selectedFloor]);

  return (
    <div className="flex h-screen w-full bg-[#f8fafc] text-slate-900 overflow-hidden select-none antialiased" style={{ fontFamily: "'Sora', sans-serif" }}>
      <style>{`@import url('https://fonts.googleapis.com/css2?family=Sora:wght@100;200;300;400;500;600;700;800&display=swap');`}</style>

      {/* SIDEBAR */}
      <aside className="w-24 bg-white border-r border-slate-200 flex flex-col items-center py-8 z-20">
        <div className="flex-1 flex flex-col gap-6 w-full items-center">
          <button 
            onClick={() => setActiveTab('master_logs')} 
            className={`w-12 h-12 rounded-xl flex items-center justify-center transition-all ${activeTab === 'master_logs' ? 'bg-slate-950 text-white shadow-lg shadow-slate-200' : 'text-slate-300 hover:text-slate-600'}`}
          >
            <Home size={20} />
          </button>
          
          <button 
            onClick={() => setActiveTab('sections')} 
            className={`w-12 h-12 rounded-xl flex items-center justify-center transition-all ${activeTab === 'sections' ? 'bg-slate-950 text-white shadow-lg shadow-slate-200' : 'text-slate-300 hover:text-slate-600'}`}
          >
            <Layers size={20} />
          </button>

          <div className="w-8 h-[1px] bg-slate-100 my-2" />

          {/* Floor Fast-Switch */}
          <div className="flex flex-col gap-2.5 overflow-y-auto no-scrollbar pb-10 w-full items-center px-4">
            {floors.map((f) => {
              const hasIssue = getFloorStatus(f.id);
              const isSelected = selectedFloor === f.id;
              
              return (
                <button
                  key={f.id}
                  onClick={() => { setSelectedFloor(f.id); setActiveTab('sections'); }}
                  className={`w-10 h-10 rounded-lg flex items-center justify-center transition-all text-[10px] font-extrabold border ${
                    hasIssue 
                      ? isSelected 
                        ? "bg-red-600 border-red-600 text-white shadow-md shadow-red-100" 
                        : "bg-red-50 border-red-200 text-red-600"
                      : isSelected 
                        ? "bg-emerald-600 border-emerald-600 text-white shadow-md shadow-emerald-100" 
                        : "bg-emerald-50 border-emerald-100 text-emerald-600 hover:border-emerald-300"
                  }`}
                >
                  {f.id}
                </button>
              );
            })}
          </div>
        </div>

        <button className="w-12 h-12 rounded-xl flex items-center justify-center text-slate-300 hover:text-slate-600 transition-all mt-auto">
          <Settings size={20} />
        </button>
      </aside>

      {/* MAIN VIEWPORT */}
      <main className="flex-1 flex flex-col relative overflow-hidden">
        <header className="h-16 border-b border-slate-200 flex items-center justify-between px-8 bg-white/80 backdrop-blur-md z-10">
          <h1 className="text-[10px] font-extrabold tracking-[0.2em] uppercase text-slate-400">
            {activeTab === 'master_logs' ? "Central Command" : `Floor ${selectedFloor}`}
          </h1>
          <div className="flex items-center gap-4">
            <div className="text-right">
              <p className="text-[11px] font-bold text-slate-900">{user.name}</p>
              <p className="text-[9px] text-slate-400 font-bold uppercase tracking-widest">{user.role}</p>
            </div>
            <div className="w-9 h-9 rounded-xl bg-slate-50 flex items-center justify-center border border-slate-200">
              <User size={16} className="text-slate-600" />
            </div>
          </div>
        </header>

        <div className="flex-1 overflow-y-auto p-8 lg:p-12 no-scrollbar">
          {activeTab === 'master_logs' && (
            <div className="max-w-6xl mx-auto space-y-16 animate-in fade-in slide-in-from-bottom-4 duration-700">
              <div className="flex flex-col lg:flex-row items-center justify-center gap-12 lg:gap-24 py-12 bg-white rounded-[3.5rem] shadow-sm border border-slate-100">
                <div className="relative w-80 h-80 flex items-center justify-center">
                  <svg className="absolute inset-0 w-full h-full -rotate-90">
                    <circle cx="160" cy="160" r="145" stroke="currentColor" strokeWidth="6" fill="transparent" className="text-slate-100" />
                    <circle 
                      cx="160" cy="160" r="145" 
                      stroke="currentColor" strokeWidth="12" fill="transparent" 
                      strokeDasharray={2 * Math.PI * 145}
                      strokeDashoffset={2 * Math.PI * 145 * (1 - metrics.health / 100)}
                      strokeLinecap="round"
                      className={`transition-all duration-1000 ease-out ${metrics.health > 80 ? 'text-emerald-500' : 'text-red-500'}`}
                    />
                  </svg>
                  <div className="text-center z-10">
                    <p className="text-[11px] font-extrabold uppercase tracking-[0.4em] text-slate-400 mb-2">Integrity</p>
                    <h2 className="text-8xl font-extrabold tracking-tighter leading-none text-slate-900">{metrics.health}%</h2>
                    <div className={`mt-5 inline-flex items-center gap-2 px-5 py-2 rounded-full border text-[10px] font-extrabold uppercase tracking-widest ${metrics.health > 80 ? 'bg-emerald-50 border-emerald-100 text-emerald-600' : 'bg-red-50 border-red-100 text-red-600 animate-pulse'}`}>
                      {metrics.health > 80 ? <CheckCircle2 size={12}/> : <AlertTriangle size={12}/>}
                      {metrics.health > 80 ? 'Stable' : 'Critical'}
                    </div>
                  </div>
                  <div className={`absolute top-4 right-4 w-14 h-14 text-white rounded-full shadow-2xl flex items-center justify-center border-4 border-white ${metrics.urgent > 0 ? 'bg-red-600 animate-bounce' : 'bg-emerald-500'}`}>
                    <span className="text-sm font-extrabold">{metrics.urgent}</span>
                  </div>
                </div>

                <div className="flex flex-col gap-8">
                  <div className="flex items-center gap-6 p-4 rounded-3xl hover:bg-slate-50 transition-all group">
                    <div className="w-16 h-16 rounded-2xl bg-red-50 flex items-center justify-center text-red-600 group-hover:scale-110 transition-transform">
                      <Zap size={24} />
                    </div>
                    <div>
                      <h4 className="text-3xl font-extrabold text-slate-900">{metrics.urgent}</h4>
                      <p className="text-[10px] font-bold uppercase tracking-widest text-slate-400 mt-0.5">Urgent Breaches</p>
                    </div>
                  </div>
                  <div className="flex items-center gap-6 p-4 rounded-3xl hover:bg-slate-50 transition-all group">
                    <div className="w-16 h-16 rounded-2xl bg-orange-50 flex items-center justify-center text-orange-600 group-hover:scale-110 transition-transform">
                      <Activity size={24} />
                    </div>
                    <div>
                      <h4 className="text-3xl font-extrabold text-slate-900">{metrics.total}</h4>
                      <p className="text-[10px] font-bold uppercase tracking-widest text-slate-400 mt-0.5">Pending Flow</p>
                    </div>
                  </div>
                  <div className="flex items-center gap-6 p-4 rounded-3xl hover:bg-slate-50 transition-all group">
                    <div className="w-16 h-16 rounded-2xl bg-blue-50 flex items-center justify-center text-blue-600 group-hover:scale-110 transition-transform">
                      <Cpu size={24} />
                    </div>
                    <div>
                      <h4 className="text-3xl font-extrabold text-slate-900">98%</h4>
                      <p className="text-[10px] font-bold uppercase tracking-widest text-slate-400 mt-0.5">System Uptime</p>
                    </div>
                  </div>
                </div>
              </div>

              <div className="space-y-6">
                <div className="flex items-center justify-between">
                  <h3 className="text-[11px] font-extrabold uppercase tracking-[0.4em] text-slate-400">Live Diagnostics Feed</h3>
                  <button onClick={() => setIsReporting(true)} className="bg-slate-950 text-white px-8 py-3.5 rounded-2xl font-extrabold text-[10px] uppercase tracking-[0.1em] hover:bg-slate-800 transition-all shadow-xl shadow-slate-200 active:scale-95">
                    + Log New Issue
                  </button>
                </div>
                <div className="grid grid-cols-1 lg:grid-cols-2 gap-5">
                  {mockIssues.map(issue => (
                    <div key={issue.id} className="group bg-white border border-slate-200 p-7 rounded-[2.5rem] hover:border-slate-400 transition-all flex items-center justify-between cursor-pointer">
                      <div className="flex items-center gap-6">
                        <div className={`w-16 h-16 rounded-2xl flex items-center justify-center ${issue.status === 'Ongoing' ? 'bg-red-50 text-red-600' : 'bg-emerald-50 text-emerald-600'}`}>
                          {issue.status === 'Ongoing' ? <ShieldAlert size={24}/> : <CheckCircle2 size={24}/>}
                        </div>
                        <div>
                          <h5 className="font-extrabold text-lg text-slate-900 leading-tight">{issue.desc}</h5>
                          <div className="flex items-center gap-3 mt-1.5">
                            <span className="text-[10px] font-extrabold uppercase text-slate-400 tracking-widest">{issue.dept}</span>
                            <span className="text-[10px] font-extrabold uppercase bg-slate-50 px-2.5 py-1 rounded-lg text-slate-500 tracking-widest border border-slate-100">FL {issue.floor}</span>
                          </div>
                        </div>

                  ))}
                </div>
              </div>
            </div>

              <div className="flex items-end justify-between border-b border-slate-200 pb-12">
                <div>
                  <h2 className="text-7xl font-extrabold tracking-tighter uppercase leading-none text-slate-900">Floor {selectedFloor}</h2>
                  <p className="text-slate-400 font-extrabold text-[11px] uppercase tracking-[0.4em] mt-6">Diagnostic Matrix</p>
                </div>
              </div>

              <div className="grid grid-cols-2 sm:grid-cols-4 lg:grid-cols-8 gap-5">
                {floorRooms.length > 0 ? (
                  floorRooms.map((room) => (
                    <div 
                      key={room.id}
                      className={`h-32 rounded-3xl border-2 transition-all flex flex-col items-center justify-center gap-1.5 ${
                        room.hasIssue 
                        ? 'bg-red-50 border-red-200 text-red-600 shadow-lg shadow-red-50/50' 
                        : 'bg-emerald-50 border-emerald-100 text-emerald-600 hover:border-emerald-300'
                      }`}
                    >
                      <span className={`text-[9px] font-extrabold uppercase tracking-widest ${room.hasIssue ? 'text-red-400' : 'text-emerald-500/60'}`}>Room</span>
                      <span className="text-2xl font-extrabold tracking-tighter">{room.id}</span>
                      {room.hasIssue ? <AlertTriangle size={16} className="mt-1 animate-pulse" /> : <CheckCircle2 size={16} className="mt-1 opacity-30" />}
                    </div>
                  ))
                ) : (
                  floors.find(f => f.id === selectedFloor).areas.map((area, idx) => {
                    const areaHasIssue = mockIssues.some(iss => iss.area === area && iss.status === 'Ongoing');
                    return (
                      <div key={idx} className={`col-span-2 h-44 rounded-[2.5rem] p-8 flex flex-col justify-between border-2 transition-all cursor-pointer ${
                        areaHasIssue 
                        ? 'bg-red-50 border-red-200 text-red-600' 
                        : 'bg-emerald-50 border-emerald-100 text-emerald-700 hover:border-emerald-300'
                      }`}>
                        <div className="flex justify-between items-start">
                          <MapPin size={24} className={areaHasIssue ? 'text-red-500 animate-pulse' : 'text-emerald-500/40'} />
                          {!areaHasIssue && <CheckCircle2 size={20} className="text-emerald-500/30" />}
                        </div>
                        <div>
                          <p className="text-lg font-extrabold uppercase tracking-tight mb-1.5">{area}</p>
                          <p className={`text-[10px] font-extrabold uppercase tracking-widest ${areaHasIssue ? 'text-red-400' : 'text-emerald-600/50'}`}>
                            {areaHasIssue ? 'Breach Detected' : 'Operational'}
                          </p>
                        </div>
                      </div>
                    );
                  })
                )}
              </div>

              {currentFloorIssues.length > 0 && (
                <div className="grid grid-cols-1 md:grid-cols-2 gap-8 pt-12 border-t border-slate-200">
                  {currentFloorIssues.map(issue => (
                    <div key={issue.id} className="bg-red-50 border-2 border-red-100 p-10 rounded-[3rem] relative overflow-hidden group">
                      <div className="absolute top-0 right-0 p-8 opacity-5 group-hover:opacity-10 transition-opacity">
                        <AlertTriangle size={120} />
                      </div>
                      <div className="flex justify-between items-start mb-8 relative z-10">
                        <span className="text-[11px] font-extrabold text-white bg-red-600 px-4 py-1.5 rounded-full uppercase tracking-widest">{issue.priority}</span>
                        <span className="text-[10px] font-bold text-slate-400 uppercase">{issue.time}</span>
                      </div>
                      <h4 className="text-3xl font-extrabold mb-2 uppercase tracking-tighter text-slate-900 relative z-10">{issue.desc}</h4>
                      <p className="text-[11px] font-extrabold text-red-600 uppercase tracking-[0.2em] mb-10 relative z-10">{issue.area}</p>
                      <button className="w-full bg-slate-950 text-white py-5 rounded-2xl font-extrabold text-[11px] uppercase tracking-widest hover:bg-slate-800 transition-all active:scale-[0.98] shadow-xl shadow-slate-200 relative z-10">
                        Acknowledge Breach
                      </button>
                    </div>
                  ))}
                </div>
              )}
            </div>
          )}
        </div>
      </main>

      {/* MODAL */}
      {isReporting && (
        <div className="fixed inset-0 z-[100] flex items-center justify-center p-6 bg-slate-950/20 backdrop-blur-xl animate-in fade-in duration-300">
          <div className="bg-white w-full max-w-xl rounded-[3.5rem] border border-slate-200 p-12 shadow-2xl animate-in zoom-in-95 duration-300">
            <div className="flex items-center justify-between mb-10">
              <h2 className="text-3xl font-extrabold uppercase tracking-tight text-slate-900">Log Diagnostic</h2>
              <button onClick={() => setIsReporting(false)} className="w-12 h-12 flex items-center justify-center rounded-full bg-slate-50 text-slate-400 hover:text-slate-900 transition-all"><Plus className="rotate-45" size={28} /></button>
            </div>
            <textarea 
              placeholder="Describe the anomaly..." 
              className="w-full bg-slate-50 border border-slate-100 rounded-[2rem] p-8 text-slate-900 text-base font-medium focus:outline-none focus:ring-2 focus:ring-slate-900/5 focus:border-slate-300 min-h-[180px] mb-10" 
            />
            <button className="w-full py-6 bg-slate-950 text-white rounded-[2rem] font-extrabold uppercase text-[12px] tracking-widest active:scale-[0.97] transition-all shadow-2xl shadow-slate-950/20">
              Broadcast to Command
            </button>
          </div>
        </div>
      )}
    </div>
  );
};

export default App;
